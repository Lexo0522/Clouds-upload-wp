# 测试连接功能使用说明

## 功能概述

测试连接功能允许你在保存配置之前验证云存储设置是否正确，避免因配置错误导致上传失败。

## 功能特性

✅ **完整的测试流程**
- 配置验证（检查必填字段）
- 存储驱动创建
- 上传测试
- 文件访问测试
- 删除测试
- 测试文件自动清理

✅ **详细的错误信息**
- 每一步都有详细的日志
- 失败时会给出具体的错误原因
- 提供修复建议

✅ **友好的用户界面**
- 一键测试按钮
- 实时显示测试进度
- 美观的结果展示

## 使用步骤

### 1. 访问设置页面
进入 WordPress 后台 → 设置 → WP Multi Cloud Storage

### 2. 填写配置信息
在保存设置之前，先填写以下信息：
- **Access Key**: 七牛云 AK
- **Secret Key**: 七牛云 SK
- **Bucket**: 存储空间名称
- **加速域名**: 七牛云绑定的访问域名（必须以 http:// 或 https:// 开头）
- **上传接口**: 可选，默认为华东地区上传地址

### 3. 点击测试连接
点击页面顶部的「测试连接」按钮，系统会自动执行以下测试：

```

  - URL: https://cdn.example.com/wpmcs-test-20260324123456.txt
步骤 4: ✓ 文件访问测试成功
步骤 5: ✓ 删除测试成功
✓ 测试文件已清理
```

## 测试内容说明

### 1. 配置验证
检查必填字段是否填写，格式是否正确。

**验证项**：
- Access Key 不能为空
- Secret Key 不能为空
- Bucket 不能为空
- 加速域名格式正确（必须包含 http:// 或 https://）
- 上传接口格式正确

### 2. 上传测试
创建一个临时测试文件并上传到云存储。

**测试内容**：
- 创建临时测试文件
- 使用配置的上传接口上传
- 验证上传返回的结果

### 3. 文件访问测试
通过 HTTP 请求访问上传的测试文件，验证：
- 文件可以正常访问（HTTP 200）
- 文件内容正确

### 4. 删除测试
删除上传的测试文件，验证删除权限。

### 5. 自动清理
测试完成后，自动删除所有测试文件，不会占用存储空间。

## 常见问题

### Q: 测试失败提示"上传接口错误"
**A**: 检查上传接口是否正确：
- 华东：`https://upload.qiniup.com`
- 华北：`https://upload-z1.qiniup.com`
- 华南：`https://upload-z2.qiniup.com`
- 北美：`https://upload-na0.qiniup.com`
- 东南亚：`https://upload-as0.qiniup.com`

### Q: 测试失败提示"域名无法访问"
**A**: 检查加速域名配置：
- 确保域名已正确解析到七牛云
- 确保域名使用 http:// 或 https:// 开头
- 确保域名没有多余的路径或参数

### Q: 测试失败提示"权限不足"
**A**: 检查 Access Key 和 Secret Key：
- 确保密钥正确无误
- 确保密钥有上传和删除权限
- 确保密钥对应的 Bucket 正确

### Q: 测试成功但实际使用失败
**A**: 可能的原因：
- 测试时配置正确，但保存时配置被修改
- 实际上传的文件类型或大小有限制
- 服务器环境有其他限制（如防火墙）

## 技术实现

### 核心类：`WPMCS_Connection_Tester`

**主要方法**：
- `run_full_test()`: 运行完整测试
- `validate_config()`: 验证配置
- `test_upload()`: 测试上传
- `test_file_access()`: 测试文件访问
- `test_delete()`: 测试删除
- `quick_test()`: 快速测试（仅验证配置）

### AJAX 接口

**端点**: `wp_ajax_wpmcs_test_connection`

**请求参数**：
```
action: wpmcs_test_connection
nonce: [安全令牌]
```

**响应格式**：
```json
{
  "success": true,
  "data": {
    "message": "连接测试成功！",
    "details": [
      "✓ 配置验证通过",
      "✓ 存储驱动创建成功",
      "..."
    ]
  }
}
```
